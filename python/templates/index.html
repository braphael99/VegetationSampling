Hello, Welcome to Vegetation Sampling!

The purpose of this web app is to perform a few things:
  • Display the relationship between our target variable (height) and our explanatory variables (Circumference, Diameter, Basal Area, and if the specimen is dead or not). The correlation coefficient and analysis will be included on these pages.
  • Allow the user to create new entries into our database.
  • Allow the user to perform hypothesis testing.

About the Dataset:


This dataset was pulled from Kaggle, posted by user Miguel Rojas. This dataset is being used under the Attribution 4.0 International (CC BY 4.0) licensing rights. Here is what the poster had to say about this dataset: "This data contains information about a vegetation sampling made in 'Cerro El Volcán' located in Caracas, Venezuela, covered by a dense forest classified as 'Premontane Moist Forest' by Holdridge. The sampling was carried out as part of the 'Ecology Laboratory I' course from the biology carrer at Central University of Venezuela in the period I-2023, in an area of 400 m². Data presented here corresponds to synthetic generated data based on real observations, so it preserves the original characteristics of the dataset." This dataset originally had 8 variables, but we will be manipulating 6, with height as our target variable. Here is a link to the original Kaggle dataset: Original Vegetation Sampling Dataset.
