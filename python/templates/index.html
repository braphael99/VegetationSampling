Hello, Welcome to Vegetation Sampling!

The purpose of this web app is to perform a few things:
  • Display the relationship between our target variable (height) and our explanatory variables (Circumference, Diameter, Basimetric Area, and if the specimen is dead or not). The correlation coefficient and analysis will be included on these pages.
  • Allow the user to create new entries into our database. (WIP)
  • Allow the user to perform hypothesis testing.
  • Perform predictive statistics on the sample data using a linear regression.
